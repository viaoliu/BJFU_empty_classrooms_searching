# 北京林业大学空课表查询工具

> 因“零课表”停止运营，图书馆不舒服，我始终没找到趁手的空教室查询工具，于是自己动手“Vibe Coding”了这个小程序。

---

## 📦 使用方式

### 方式一：直接运行 EXE（推荐）
- 无需安装 Python，无需输入学号密码
- 下载 `dist` 文件夹中的 `空教室查询.exe`，双击即可使用
- 内置免登录逻辑，开箱即用

### 方式二：运行 Python 源码
1. 确保已安装 Python 3.x 及所需依赖
2. 打开 `main.py`，将**第 342、343 行**的学号和密码修改为自己的（本地运行，无服务器上传，隐私安全）
3. 执行 `python main.py` 启动程序

### 方式三：一键配置（可选）
- 若想定制功能或打包自己的版本，可运行 `set_up.py` 在修改代码后进行打包

---

## ✨ 功能特点

- **免登录**：EXE 版本已内置认证，无需每次输入账号密码
- **无校园网限制**：自动与北林 VPN 交互，校外网络也能正常查询
- **极致快捷**：告别教务系统或 APP 的繁琐层级，一键展示当前所有空闲教室，一目了然
